Function of segment_parser.py:
This program takes in a data file from a segment folder and outputs a text file with data about each url that nutch tried to fetch
as well as compounds statistics on the mime types encountered, images count and nutch statuses count on a particular segment data file.
The statistics are shown at the end of the output text file after the following:
****************************************STATISTICS******************************************



To run segment_parser.py type in terminal:

python segment_parser.py <path/to/segments/data> <outputfile>
